# Sprout Social - Quick Start Guide

## 🚀 První Kroky

### 1. Přihlášení a Setup
```
1. Jít na https://sprout.social
2. Vytvořit nový účet nebo se přihlásit
3. Vybrat plán (Business, Agency, Enterprise)
4. Propojit první sociální síť
```

### 2. Propojení Sociální Sítě

#### Facebook
```
1. Settings → Integrations → Facebook
2. Click "Connect Account"
3. Schválení OAuth
4. Vybrat stránky k synchronizaci
5. Udělení oprávnění
```

#### Instagram
```
1. Settings → Integrations → Instagram
2. Propojit přes Facebook
3. Vybrat Business account
4. Synchronizace účtu
```

#### Twitter/X
```
1. Settings → Integrations → Twitter
2. Autentizace přes OAuth
3. Schválení přístupu
```

### 3. Vytvoření Prvního Příspěvku

**Krok 1:** Jít na Content → Composer
```
┌─────────────────────────────────┐
│ Create New Post                 │

└─────────────────────────────────┘
```

**Krok 2:** Přidat média
```
Click: Upload Images/Video
Vybrat soubor z počítače
Nakrátko: Drag & Drop
```

**Krok 3:** Vybrat sítě
```
☑️ Facebook
```

**Krok 4:** Zveřejnit nebo naplánovat
```
Publish Now      → Okamžité publikování

Save as Draft    → Uložit jako koncept
```

## 📅 Tipů pro Efektivní Správu

### Content Calendar Best Practices
```
1. Plánujte 2 týdny dopředu
2. Používejte Best Time to Post
3. Publikujte 4-7 krát týdně
4. Mixujte typy obsahu (video, carousel, text)
5. Reagujte na komentáře do 2 hodin
```

### Optimal Posting Times
```
Facebook: 1-3 PM (pracovní dny)
Instagram: 6-9 AM, 12-2 PM, 7-9 PM
Twitter: 8-10 AM, 5-6 PM
LinkedIn: 7-9 AM, 12 PM (ÚT-ČTV)
```

### Hashtag Strategy
```
- 5-10 hashtags per post
- Mix popular + niche hashtags
- Umístění: konec caption (Instagram), separat (Twitter)
- Sledujte performance
```

## 🎯 Měření Výkonu

### Key Metrics to Track
```
1. Reach & Impressions
   → Jak mnoho lidí vidí váš obsah

2. Engagement Rate
   → (Likes + Comments + Shares) / Reach × 100

3. Click-through Rate
   → Traffic ze sociálních sítí na váš web

4. Follower Growth
   → Nový followers za měsíc

5. Sentiment
   → Pozitivní vs. negativní komentáře
```

### Monthly Review Checklist
```
☐ Proveřit top 5 best performing posts
☐ Porovnat metriky s minulým měsícem
☐ Analyzovat audience demographics
☐ Přehodnotit content strategy
☐ Nastavit cíle pro příští měsíc
```

## 🤝 Team Collaboration

### Přidat Týmové Členy
```
Settings → Team → Add Member
│
├── Email: team@company.com
├── Role: Manager
│   └── (Admin, Manager, Member, Viewer)
└── Send Invite
```

### Approval Workflow
```
1. Team member: Vytvoří post → Submit for Approval
2. Manager: Dostane notifikaci
3. Manager: Review → Approve / Request Changes
4. Team member: Revize → Schedule/Publish
5. Post: Publikován!
```

## 🆘 Troubleshooting

### Account Connection Issue
```
Řešení:
1. Check OAuth token expiration
2. Re-authenticate account
3. Verify API permissions
4. Contact platform support
```

### Missing Analytics Data
```
Řešení:
1. Počkejte 24 hodin na synchronizaci
2. Obnovte stránku
3. Zkontrolujte, zda je účet stále připojen
4. Ověřte oprávnění API
```

### Post Not Publishing
```
Řešení:
1. Zkontrolujte chyby v logu
2. Ověřte, zda je obsah v souladu s pravidly
3. Zkontrolujte připojení k internetu
4. Opakujte publikování
```
